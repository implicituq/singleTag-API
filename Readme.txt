
Dependencies - Dependencies required for the project 
    -> npm install express
    -> npm install superagent
    -> npm install ejs

How to's - Perform the following before starting the App, as only a test API Link has been provided which is not intended for any promotional uses 
    -> The API link you need to fetch data from has to be feeded in the file 'apiControl.js' in the folder 'Controller'
    -> Paste your required API link in the indicated area in the apiControl.js file (Instructions in the form of comments are provided)

Execution - it runs on the server -> "http://localhost:4000/"                                                
            the command start server -> npm start
